Project Overview

This Digital Literacy Project aims to develop an understanding of essential digital skills required for academic and professional growth. As a Student Digital Ambassador, the objective of this project is to create awareness among peers about the effective use of digital tools, safe online practices, and professional communication.

The project is divided into five tasks, each focusing on a specific aspect of digital literacy. These tasks include creating an infographic, building a digital portfolio, exploring coding and collaboration platforms, understanding email etiquette, and studying cybercrime awareness.

Task 1 – Infographic

A digital literacy awareness infographic was created using Canva. The infographic covers key topics such as the meaning of digital literacy, useful digital tools for students, safe internet practices, and email etiquette. The design is structured with clear headings, icons, and minimal text to ensure readability and visual clarity.

Folder: task-1-Presentation

Task 2 – Digital Portfolio

Profiles were created and updated on professional platforms such as GitHub and LinkedIn. A GitHub profile README was developed including basic personal and academic details. The LinkedIn profile was updated with educational information. Screenshots of all profiles have been included.

Folder: task-2-Portfolio

Task 3 – Platforms

A beginner-level coding problem was solved on HackerRank to understand basic problem-solving skills. Additionally, a Digital Literacy Awareness Quiz was created using Google Forms, consisting of multiple-choice and short-answer questions. The form is linked to a Google Sheet to collect responses.

Task 5 – Cybercrime Awareness

A case study on UPI fraud was prepared, explaining how such frauds occur, who is targeted, and their consequences. A prevention checklist was also created, including safety tips and reporting methods.
